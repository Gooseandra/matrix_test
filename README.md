## Использование

### Рисование фигур

matrix_draw N M line x1 y1 x2 y2 [фон] [символ]
matrix_draw N M ellipse cx cy rx ry [фон] [символ]
matrix_draw N M rect x1 y1 x2 y2 [угол] [фон] [символ]

markdown
Копировать код

- `N` — количество строк (высота матрицы)  
- `M` — количество столбцов (ширина матрицы)  
- `x1 y1 x2 y2` — координаты для линии или прямоугольника  
- `cx cy rx ry` — центр и радиусы для эллипса  
- `угол` — угол поворота прямоугольника в градусах (необязательный, по умолчанию 0)  
- `фон` — символ фона (необязательный, по умолчанию `.`)  
- `символ` — символ фигуры (необязательный, по умолчанию `#`)  

#### Примеры

- Линия:  
matrix_draw 20 40 line 1 1 18 38

### Загрузка матрицы из файла

matrix_draw load <путь_к_файлу>

markdown
Копировать код

- Загружает ранее сохранённую матрицу и выводит её в консоль.  
- Не создаёт новую фигуру, только отображает содержимое файла.  

#### Пример

matrix_draw load matrix.txt

markdown
Копировать код

### Сохранение

- При рисовании фигур матрица автоматически сохраняется в файл `matrix.txt`.  
- Для повторного использования можно загрузить её командой `load`.

### Примечания

- Координаты `(0,0)` находятся в верхнем левом углу матрицы.  
- Для более квадратного вида используйте символы `█`, `■` или дублируйте символы при выводе.  
- Можно изменять фон и символ фигуры, передавая соответствующие аргументы.
